my commands for terminal
push - отправить изменения на удалённый репозиторий
commit - зафиксировать изменения
status - просмотр изменений
add - добавление файла в отслеживание гита
switch - переключение между ветками
branch - просмотр веток
nano - редактирование текстового документа
touch - создание нового файла через терминал
